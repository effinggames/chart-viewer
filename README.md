# Chart-viewer

Companion charting app to stock-trader. Easy interface to see previous results and historical performance.    

Using Play Framework, Quill, Node/Npm/Gulp, and Highcharts.

To run:  
`npm install`  
`npm run build`  
Fill out env vars.  
Then `sbt run`

Env Vars:  
`DATABASE_USER`  
`DATABASE_PASSWORD`  
`DATABASE_NAME`  
`DATABASE_URL`  
`DATABASE_PORT`
